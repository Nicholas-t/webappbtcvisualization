BITCOIN BLOCKCHAIN VISUALIZER

CSE 303

How to run :
- Be sure to run the rest API by running 'node rest' in command prompt.
- put the height of the block you want to visualize.
- the "heigh end" entry is optional. Add a number if you want to visualize
an interval of blockheight.
- the plot button will open a full screen visualization of the transaction in the block.
- right click on the nodes or links and it will redirect you to the address details or
the transaction details respectively.
- press "BACK" on the top left or "Esc" twice if you want to exit the fullscreen.
